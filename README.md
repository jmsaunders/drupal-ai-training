# Drupal AI Training

Open-source, session-based curriculum for practical AI integration in Drupal. This repository provides modular lesson content, labs, and references you can reuse in workshops, camps, and internal trainings. Times, dates, and personal bios have been removed to keep it generic.

## Objectives
- Understand foundational AI concepts in a Drupal context.
- Install and configure DrupalAI with real-world use cases.
- Build editorial automations using prompt-based workflows.
- Apply responsible AI practices around accessibility, bias, and transparency.

## How to Use
- Deliver as a single-day workshop or split into modules.
- Clone and adapt the sessions to your audience.
- Pair sessions with the matching exercises for hands-on practice.
