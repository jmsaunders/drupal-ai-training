## CKEditor integration
Enable the AI toolbar and sidebar to unlock:
- Reformat HTML
- Fix spelling
- Summarise text
- Tone controls (via taxonomy descriptions)

## Suggested flow
1. Create a Tone vocabulary (e.g., Happy, Sad, Angry, Mine, “David Attenborough”).
2. In text formats, enable AI → Tone and select the vocabulary; use term descriptions as tone guidance.
3. Turn on AI Content Suggestions (Readability, Summarise, Suggest Title).

## Assistants for site building
- Create an AI Assistant with Agents enabled.
- Place AI DeepChat in a convenient admin region.
- Example prompts: create vocabularies/terms, scaffold content, generate test data.

> Guardrail: keep generated content in review fields or non-public content types until approved.
