Goal: Rewrite titles and bodies to a consistent reading level using Automators.

Steps:
1. Create fields: Simplified Title (formatted text) and Simplified Body (long formatted).
2. Attach Automators to each field.
3. Use a title prompt that outputs <h2> only; use a body prompt that outputs strict JSON with limited tags.
4. Set temperature to 0–0.1 and choose your provider/model.
5. Save a node and verify results in the simplified fields.
6. Only publish simplified fields after editor review.
