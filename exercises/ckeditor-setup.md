1. Go to Configuration → Content Authoring → Text Formats and Editors.
2. Drag the AI icon into the toolbar.
3. Enable: Reformat HTML, Fix Spelling, Summarise, Tone.
4. Enable AI Content Suggestions: Readability, Summarise Text, Suggest Title.
5. Create a test node and try the tools.
